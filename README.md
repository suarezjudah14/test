# Student Productivity Portfolio homepage

This project is a multipage web application built using Python and Streamlit. It serves as a simple student portfolio and productivity app. The app includes several pages such as Home, Profile, Study Tracker, Grade Calculator, and Contact.

## Project Description

The Student Productivity Portfolio App is designed to showcase basic Streamlit skills through a clean and organized multipage application. It includes personal information, study tracking features, a simple grade calculator, and a contact form.

This project demonstrates how Streamlit can be used to create interactive web applications with simple Python code.

## Features

- Multipage navigation using Streamlit sidebar
- Home page with welcome message and user input
- Profile page with personal background and learning goals
- Study Tracker page with subject selection, study hours slider, and weekly study plan
- Grade Calculator page for computing final grades
- Contact page with a simple form
- Clean and beginner-friendly layout
- Interactive components such as buttons, text inputs, sliders, checkboxes, and data tables
